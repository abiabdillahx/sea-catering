## ⬇️ Run the project
Run on the local server:
```bash
# clone the repo 
git clone https://github.com/abiabdillahx/sea-catering.git
cd sea-catering

# install the dependencies
npm install

# start the server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
